# BookNook

Welcome to BookNook! This repository hosts the source code for a website dedicated to books, literature, and reading enthusiasts.

## Overview

BookNook is a platform designed to connect book lovers, offer a curated selection of books, and provide engaging content related to reading and literature. Whether you're looking for book recommendations, author insights, or a community of fellow readers, MyBookWebsite has something for everyone.

## Features

- Browse a diverse catalog of books across various genres.
- Discover bestsellers.
- Browse your favourite books using the browse books feature.
- Get a brief information of the books with a preview link for each book to its Google Books site.

## Technologies Used

- HTML/CSS: Frontend structure and styling.
- JavaScript: Dynamic content and interactivity.
- Tailwind CSS: Utility-first CSS framework for styling.
- Google Books API: For fetching the details of the books.
- New York Times API: For fetching the list of bestseller books from the New York Times.
